<h3>Error Description</h3>
The XML request is not formatted properly.

<h3>Example</h3>
<pre>
400 Bad Request
</pre>

<h3>How to Resolve</h3>
Confirm that the format of the XML request is set properly and includes all expected opening and closing tags.
