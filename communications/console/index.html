NpaNxxToPCode

Try it now!

This method returns the PCode for the specified NpaNxx value. If the NpaNxx value is invalid or has no EZtax jurisdiction the return will be NULL. Returns the PCode that corresponds to an NpaNxx value.


API ENDPOINT
POST https://eztaxasp.billsoft.com/EZTaxWebService/EZTaxWebService.svc?NpaNxxToPCode
HEADERS
Content-Type: application/json

DescriptionThis method returns the PCode for the specified NpaNxx value. If the NpaNxx value is invalid or has no EZtax jurisdiction the return will be NULL. Returns the `PCode` that corresponds to an `NpaNxx` value.
Endpointhttps://eztaxasp.billsoft.com/EZTaxWebService/EZTaxWebService.svc?NpaNxxToPCode
HTTP Methodpost
